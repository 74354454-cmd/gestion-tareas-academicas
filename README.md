# 🎓 Sistema de Gestión de Tareas Académicas
**Producto Académico N° 1 - Construcción de Software**

## 📝 Descripción del Proyecto
Esta aplicación es una herramienta sencilla para que los estudiantes de Ingeniería de Sistemas puedan organizar sus entregas, permitiendo registrar tareas, asignar prioridades y validar fechas límite[cite: 12, 19].

## ⚙️ Tecnologías y Metodologías Aplicadas
* **Metodología Ágil:** Se implementó una iteración (Sprint) para el desarrollo de las funciones principales[cite: 13, 20].
* **IA Generativa:** Se utilizó Inteligencia Artificial para la generación de la estructura del código y optimización de funciones[cite: 14, 21].
* **Lenguaje:** Python 3.x.

## 🛠️ Estándares de Calidad (Rúbrica)
* **Manejo de Excepciones:** Se implementaron bloques try-except para asegurar que la aplicación sea robusta ante errores de entrada[cite: 15, 22].
* **Clean Code:** El código sigue principios de modularidad y legibilidad para facilitar su mantenimiento[cite: 16, 23].
* **Refactorización:** Se mejoró la lógica inicial para asegurar eficiencia en el manejo de datos[cite: 17, 23].
